# EmailOps

A Windows desktop application built with Python and CustomTkinter for creating Outlook draft emails, reviewing company contact information from Excel, and automating monthly report email drafts with attachments.

## Features

- Load and search company information from an Excel file
- View company contact details and all Excel columns
- Create blank Outlook draft emails with signature
- Create template-based Outlook draft emails
- Generate monthly report draft emails with attachments
- Support special attachment rules for selected folders
- Produce a local execution summary file

## Requirements

- Windows
- Microsoft Outlook installed and configured
- Python 3.10 or newer
- Local access to the synced project folder
- Excel source file available locally
